# 🎵 Music Genre Classification using Deep Learning

## 📌 Project Overview
This project predicts the genre of a music file using Deep Learning. It extracts MFCC (Mel-Frequency Cepstral Coefficients) features from audio files and classifies them into different music genres.

## 🚀 Features
- Upload MP3 audio file
- Extract MFCC features using Librosa
- Predict music genre
- Display prediction confidence

## 🎼 Supported Genres
- Blues
- Classical
- Country
- Disco
- Hip-Hop
- Jazz
- Metal
- Pop
- Reggae
- Rock

## 🛠️ Technologies Used
- Python
- TensorFlow / Keras
- Librosa
- NumPy
- Google Colab

## 📂 Project Files
- `Music_Genre_Classification.ipynb` – Main notebook
- `music_genre_model.h5` – Trained model

## ▶️ How to Run
1. Open the notebook in Google Colab.
2. Install the required libraries.
3. Upload an MP3 audio file.
4. Run all cells.
5. View the predicted music genre and confidence score.

Genre: Country
